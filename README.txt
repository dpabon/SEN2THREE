Sen2Three: Sentinel 2 Level 3 Spatio Temporal Processor 
=================================================================================

Sen2Three is a Level 3 Processor for the spatio temporal synthesis of series of
Level 1C or Level 2A input images. It is able to remove clouds and other artifacts
from input images and generates a mosaic image showing only relevant pixels.

Sen2Three works under the Umbrella of Anaconda (Python 2.7)
on the three platforms Mac OSX, Linux, and Windows 64-bit. This is mandatory! 

Please read the docs located on: http://s2tbx.telespazio-vega.de for further info.
